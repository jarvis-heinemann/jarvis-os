## рҹ”§ Setup Required (15 minutes)

### Step 1: Convex (5 min)
```bash
cd ~/.openclaw/workspace/mission-control
npm install convex
npx convex dev
```

Follow prompts to create account and project.

### Step 2: Google Drive (5 min)
1. Create folder: `Google Drive/Obsidian/Mission Control`
2. Open Obsidian вҶ’ Create vault in that folder
3. Done! Google Drive auto-syncs

### Step 3: Configure (5 min)
```javascript
// In browser console or add to app.js
const sync = new MissionControlSync();
sync.setObsidianPath('~/Google Drive/Obsidian/Mission Control');

// Initialize Convex (get URL from npx convex dev)
const convex = new Convex.ConvexClient("https://your-url.convex.cloud");
sync.initConvex(convex);
```
