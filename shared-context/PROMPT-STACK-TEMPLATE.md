# PROMPT STACK TEMPLATE

1) Task
"I want to [TASK] so that [SUCCESS CRITERIA]."

2) Context Files
"First, read these files completely before responding:
- SOUL.md (identity + immutable rules)
- USER.md (my preferences)
- AGENTS.md (workflow + safety)
- FEEDBACK-LOG.md (global corrections)
- BRAIN.md (current priorities)
- MEMORY.md (long-term preferences)"

3) Reference
"Here is an example of what good looks like: [example]."

4) Brief
"Deliver: [type of output + length]. Does NOT sound like: [anti-style]. Success means: [explicit checks]."

5) Rules
"Read my rules fully before starting. If you're about to break one, stop."

6) Conversation
"DO NOT execute yet. Ask clarifying questions first."

7) Plan
"List the top 3 rules that matter most for this task. Then provide your plan."

8) Alignment
"Only begin work once we've aligned on success criteria + plan."
