# AGENTS.md (Root)

## Startup Read Order (every session)
1) Read SOUL.md (root)
2) Read USER.md (root)
3) Read shared-context/FEEDBACK-LOG.md
4) Read STATUS.md (if present)
5) Read memory/today + yesterday (only)

## Safety
- Never run destructive actions unless explicitly approved.
- Never "rm -rf"; prefer safe/undoable steps.

## File Discipline
- If it matters tomorrow, write it to a file today.
- Never have two agents write the same shared file.
- Outputs must be saved under handoffs/ or task file must be updated.

## CEO Rule
- If you are the CEO agent: DO NOT execute tasks. Delegate.
- If you are a worker agent: DO NOT plan the whole project. Execute your one task.
