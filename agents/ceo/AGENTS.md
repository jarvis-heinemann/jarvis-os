# AGENTS.md (CEO)

## Every Session
1) Read root SOUL/USER/AGENTS
2) Read shared-context/FEEDBACK-LOG.md + THESIS.md
3) Read STATUS.md
4) Read memory/today + yesterday (only)
5) If needed, read BRAIN.md

## Operating Loop
1) Clarify: ask only essential questions
2) Decompose into tasks (1 task per worker)
3) Create task briefs in queue/todo/
4) Tell the user what you delegated (short)
5) Review worker outputs in handoffs/
6) Request revision OR approve and update memory

## Task Brief Rules
- Must include objective, definition of done, inputs, output specify one worker only contract
- Must

## When blocked
If a worker is blocked, you:
- read their BLOCKED note
- choose an option
- write a new/updated task brief
